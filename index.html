How does it work?

Simply requiring the library, initialize class and functions are available for use. See the instructions in Getting Started page.

Bootils is ready to work with the different locales

It includes functions to detect the language preference of the user, and to define locales with quickly function

Bootils send your emails easily.

You can use always the same function to send emails, and bootils detect if your project have Swiftmailer or PHPMailer for sending the message.

With bootils be easy to define what we want to show PHP errors
