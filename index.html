Welcome to Bootils!

Bootils includes over 20 functions to facilitate your work with PHP, and two extra libraries to improve some common tasks such as debugging or sending emails. Discover the details of these features

How does it work?

The easiest way is simply a require_once, but if you wish or need, you can define default values before the required file bootils file

Bootils is ready to work with the different locales

It includes functions to detect the language preference of the user, and to define locales quickly

Bootils include Kint!

Kint is a beautiful replacement for var_dump(), print_r() and debug_backtrace(). You can enable or disable it easy.

Bootils include Swiftmailer!

Swiftmailer is a powerfoul component based mailing library for PHP. You can enable or disable it easy.

Bootils send your emails easily.

It does not matter if we have Swiftmailer activated or not, you can always use the same function to send emails

Errors configuration

With bootils be easy to define what we want to show PHP errors
